## 方案一：保持原有 Pixel Architecture Art 風格（忠實重建）

### Design Movement
Pixel Art + Retro Game UI，延續現有嘅建築像素風格

### Core Principles
1. 保持原有嘅暖色調 parchment 背景 (#F2E8D5)
2. 像素字體 (Press Start 2P, Silkscreen, VT323) 配搭 Nunito body font
3. 手繪風格嘅 dashed border 連接線
4. 復古遊戲 UI 嘅 box shadow 效果

### Color Philosophy
- Primary brown: #6B4C3B (邊框、文字)
- Accent green: #3D6B4F (AGENT, 按鈕)
- Accent orange: #B85C38 (警告、PERSONA card)
- Warm gold: #C67B4B (POSITION)
- Background parchment: #F2E8D5, #EDE3D0
- Muted: #A89B8C

### Layout Paradigm
Grid-based mind map 佈局，中間 PERSONA card 連接四周 data panels

### Signature Elements
- Pixel corner decorations (金色角標)
- Dashed SVG 連接線
- Retro button 風格 (box-shadow offset)

### Interaction Philosophy
Click-to-edit inline 數值，hover outline 提示

### Animation
Smooth bar transitions (0.6s ease) for perceptual load bars

### Typography System
- Display: Press Start 2P (titles, labels)
- Data: VT323 (values, numbers)
- Body: Nunito (narratives, descriptions)
</text>

## 方案二：Modernized Pixel - 保留像素元素但加入更多現代感

### Design Movement
Neo-Pixel: 結合 pixel art 同 modern dashboard

### Core Principles
1. 更大嘅字體同更清晰嘅層次
2. 加入 glassmorphism 效果
3. 保持像素字體但增大可讀性
4. 更明確嘅 section 分隔

### Color Philosophy
同上但加入更多 contrast

### Layout Paradigm
Card-based dashboard 配 pixel 裝飾

### Signature Elements
- Frosted glass panels
- Pixel border 裝飾
- Gradient accents

### Typography System
- Display: Silkscreen (larger sizes)
- Body: Nunito (larger, more readable)
